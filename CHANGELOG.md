## [0.1.0] - 2026-06-23

### Added

- Browser-based PDF toolkit with 22+ client-side tools (merge, split, protect, convert, and more)
- Local Power Mode via FastAPI backend (OCR, PDF to Word, deep compression)
- Docker Compose setup for one-command local development
- GitHub Pages live demo with optional local backend connection
- Semantic versioning and GitHub Releases workflow
